# 学生成长画像与精准支持分析平台 - 原型演示系统

## 项目概述

本项目为「学生成长画像与精准支持分析平台」的完整可交互 HTML 原型，用于与高校用户进行需求确认和方案演示。  
平台定位为**数据分析与辅助决策应用**，面向学生本人及高校各级学生工作角色提供成长画像、群体分析、重点关注识别和个性化建议等服务。

## 原型页面清单（共 14 个）

| 文件 | 页面名称 | 功能说明 |
|------|----------|----------|

| `student-profile.html` | P2 我的成长画像 | 个人信息聚合、多维画像概览、成长标签、时间轴 |
| `student-wuyu.html` | P3 五育评价 | 五育雷达、维度切换详情、学期趋势、提升建议 |
| `student-academic.html` | P4 学业发展 | 绩点趋势、课程成绩、毕业达成、学习建议 |
| `student-remind.html` | P5 提醒与推荐 | 分类提醒、个性化建议、资源推荐、推荐依据说明 |

## 入口页面

- `index.html`：角色选择入口页，含全部14个页面的导航

## 技术实现

- 纯静态 HTML + CSS + JavaScript，无框架依赖
- `css/common.css`：统一设计系统（颜色变量、布局、组件样式）
- `js/common.js`：Tab切换、弹窗/抽屉、SVG图表（雷达/折线/柱状/饼图）、Toast提示
- `js/sidebar.js`：动态侧边栏生成器，支持4种角色切换

## 角色与视角切换

所有页面均支持通过侧边栏底部「切换视角」快速跳转至其他角色视图。

## 设计原则

1. **育人导向**：以学生成长为中心，不以管控为目的
2. **分析为主**：看总览、看趋势、看原因、看建议
3. **措辞合规**：使用"关注等级"而非"风险等级"，"支持提示"而非"预警判定"
4. **权限分级**：不同角色看不同范围和敏感度的信息
5. **可解释**：每项识别结果均有证据说明和建议方向
